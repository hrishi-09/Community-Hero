How a report moves

  1. Spot it. A citizen posts a photo/video, a short description and the exact pincode.
  2. Confirm it. Neighbours tap "I see this too" to verify it's real.
  3. Route it. It lands directly on that pincode's police-station admin dashboard, tagged Pending.
  4. Resolve it. The admin fixes it, uploads proof, flips it to Resolved — and reward points land in the reporter's account.
